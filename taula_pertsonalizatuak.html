Datu deskriptiboak

ALDAGAI NUMERIKOA ETA KATEGORIALA

  • Deskriptiboak
  • Deskriptiboak taldekatze aldagai baten arabera (Aldagaiaren datuak bi talde bakarrik osatu behar dute: Adb; Sexua (Gizonezkoa-Emakumezkoa), Gradua (HH - LH). Bestela, FITXATEGIA ZATIKATU

1. Deskriptiboak eta aldagaien arteko azterketa

ANALIZAR > ESTADÍSTICOS DESCRIPTIVOS > EXPLORAR

  • Aldagaiak :

    • Hautatu Dependent variables (X) eta Factor List (taldekatze faktorea).
    • Adibidez: X = Nota batez bestekoa, Faktorea = Sexua (Gizonezkoa/Emakumezkoa edo HH/LH).

    Gráficoak :

    • Klikatu "Gráficos" botoian eta hautatu Diagrama de Cajas (Kaxa-diagramak).
    • kaxa-diagramak datuen banaketa eta taldeen arteko diferentziak bistaratzeko erabili daitezke .

2. Taldeen arteko batezbestekoak alderatzea

ANALIZAR > COMPARAR MEDIAS > PRUEBA T PARA MUESTRAS INDEPENDIENTES

  • Aldagaiak :

    • Hautatu Variable de prueba (adib., Nota batez bestekoa).
    • Hautatu Variable de agrupación (adib., Sexua edo Gradua).
    • Definir grupoak sakatu eta zehaztu bi kategoria (adib., Gizonezkoa = 1, Emakumezkoa = 2).

    Interpretazioa :

    • SPSSek bi taldeen arteko batezbestekoak alderatuko ditu eta p-balioa emango du, hipotesia ebaluatzeko .

3. Taulak pertsonalizatuak

Ruta : ANALIZAR > TABLAS > TABLAS PERSONALIZADAS

  • Ezarpenak :

    • Gehitu aldagai nagusiak eta faktoreak taulan ordenatzeko.
    • Aukeratu estatistikoak eta grafikoak gehitzeko (batezbestekoa, desbideratze estandarra, maiztasunak, etab.).

    Interpretazioa :

    • Taula pertsonalizatuak datuak modu antolatuagoan eta ulergarriagoan bistaratzeko balio dute, bereziki aldagai anitzeko analisiak egiteko
